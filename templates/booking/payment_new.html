Comment approuver votre paiement :

📱 MTN Mobile Money :

  1. Vous recevrez un SMS de MTN
  2. Composez *126# sur votre téléphone
  3. Sélectionnez "Approuver transaction"
  4. Entrez votre code PIN MoMo
  5. Confirmez le paiement

📱 Orange Money :

  1. Vous recevrez un SMS d'Orange
  2. Composez #150# sur votre téléphone
  3. Sélectionnez "Mes transactions"
  4. Approuvez la transaction en attente
  5. Entrez votre code PIN Orange Money

📱 Airtel Money :

  1. Vous recevrez un SMS d'Airtel
  2. Composez *140# sur votre téléphone
  3. Sélectionnez "Approuver paiement"
  4. Entrez votre code PIN Airtel Money
  5. Confirmez la transaction
